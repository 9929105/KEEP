Start Collecting Data Now
Start collecting immediately. Diaries can be created using existing spreadsheets or from scratch allowing you to begin collecting data immediately.

Simple Data Management

Easily manage your data in an intuitive spreadsheet-like interface that was built with little and big data in mind. Add, edit, and delete data all from a single place.

Securely Stored

Only share what you want. Two-factor authentication and 1024-bit SSL encryption used throughout the data collection process allows your data to be safely transmitted and stored.

Run Powerful Analytics

Learn more about your data. Visualizations and analytics are automatically generated to best fit your data. You can also go in and create your own visualizations and analyses, allowing you to views data in new and powerful ways.

Privatize Shared Data

Share data privately. Differential privacy allows you to share potentially confidential information by fuzzing the data with enough randomness to mask it but still provide statistically useful analytics.
